# Service Agent Platform

An end-to-end AI service agent platform for small service businesses.

## Capstone MVP

The first vertical is a restaurant reservation assistant.

This project demonstrates:

- RAG
- Hybrid retrieval
- Tool-based booking logic
- Agent orchestration
- Evaluation
- Basic monitoring
- FastAPI

## Project Structure

app/          Application source code
data/         Local datasets
docs/         Architecture and documentation
evaluations/  Evaluation datasets and scripts
scripts/      Utility scripts
tests/        Automated tests

## Tech Stack

- Python
- FastAPI
